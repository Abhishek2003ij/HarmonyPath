#Disease Prediction Using Machine Learning
This machine learning project aims to predict diseases based on symptoms provided by the user. It utilizes three different machine learning algorithms, ensuring accurate output. The user interface is built using Tkinter for a seamless experience.

#Project Overview
The primary purpose of this system is to provide predictions for general and commonly occurring diseases that, if left unchecked, could become fatal. The system employs data mining techniques and decision tree algorithms to analyze user-inputted symptoms. It predicts the most likely disease and offers measures to prevent its progression. Additionally, the system aids healthcare professionals in analyzing disease patterns within the community.

#Key Features
Disease Prediction: The system predicts potential diseases based on user-supplied symptoms.
Data Mining: Implements preliminary data mining techniques for effective training of the machine learning models.
User-Friendly GUI: Utilizes Tkinter for an intuitive graphical user interface.
Pattern Analysis: Assists doctors in understanding disease prevalence trends in society.
